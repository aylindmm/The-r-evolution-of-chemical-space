# The (r)evolution of Chemical Space

This repository contains the source code and data required to reproduce the figures and analyses presented in the article: **"The (r)evolution of chemical space and molecular modeling: A time-resolved perspective."**

## 📂 Repository Structure

### 1. Literature Analysis
This module focuses on the evolution of scientific publications and trends within the field.
*   **`CHEMBL_analysis.Rmd`**: R Markdown script used to generate the literature analysis figures.
*   **`data/`**: Directory containing the raw files retrieved from the **Web of Science**.

### 2. ChEMBL Compounds Analysis
Analysis of chemical datasets retrieved from the [ChEMBL Download page.](https://chembl.gitbook.io/chembl-interface-documentation/downloads).

*   **`chemical_space.Rmd`**: R code for data analysis and visualization.
*   **`clean_scaffolds.py`**: Python script for scaffold processing and filtering.
*   **`CHEMBL_Dimreduction.ipynb`**: Jupyter Notebook containing the dimensional reduction workflows.
*   **`chem_universe.ipynb`**: Code for generating the time-resolved video of the chemical universe history.
